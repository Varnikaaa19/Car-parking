
# CUROPark (Streamlit)

A simple Streamlit app to show parking availability and vehicle details.

## Deploy on Streamlit Cloud
1. Push these files to your GitHub repository.
2. Go to Streamlit Cloud.
3. Click **New app**, select your repo, branch `main`, and file `app.py`.
4. Deploy and share your app link.

## Notes
- Data is stored in `data.json`. On Streamlit Cloud, changes are temporary.
- For production, use a database or API.
